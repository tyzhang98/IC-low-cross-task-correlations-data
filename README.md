# IC-low-cross-task-correlations-data

Data for: **"Why do inhibitory control tasks show low cross-task correlations? A drift diffusion modeling study with cross-site validation"**

## Directory Structure

- `Raw_Edata/` — Trial-level data from E-Prime (two sites, two time points)
- `Derived_data/` — Processed behavioral indices and HDDM parameter estimates

## Tasks

- Interference control: Stroop, Flanker, Simon
- Response inhibition: Go/No-Go, Stop-Signal, Antisaccade
